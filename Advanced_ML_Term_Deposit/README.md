# Predicting Term Deposit Subscriptions
**Course:** Advanced Machine Learning (DSC 540)  
**Tech Stack:** Python, Scikit-learn, Random Forest, XGBoost  

### Overview
Developed ML models to predict customer term deposit subscriptions with 92.78% accuracy.  
Key features include call duration, campaign response, and financial behavior.
